# landrik-ui

landrik-ui is a set of React component libraries i created for KubeSphere console.

## Requirement

- Node.js 20.9.0 ([installation with nvm](https://github.com/creationix/nvm#usage))
- Yarn 1.22.4+

## Installation

```shell
npm i @landrik-ui
```

## Example

```jsx
import React, { Component } from "react";

import { Button } from "@landrik/ui";

class Example extends Component {
  render() {
    return <Button>Button</Button>;
  }
}
```
