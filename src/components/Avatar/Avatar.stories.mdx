# Avatar

Avatars can be used to represent people or objects. It supports images or letters.

`Props` are listed at the end of this page.

[View source](https://github.com/landrik/landrik-ui/tree/main/src/components/Avatar)

## Imports

```js
import { Avatar } from "@landrik/ui";
```
