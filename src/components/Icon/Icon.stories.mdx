# Icon

Icons are imported by applying name props to the `Icon` of the desired icon

`Props` are listed at the end of this page.


[View source](https://github.com/landrik/landrik-ui/tree/main/src/components/Icon)

## Imports

```js
import { Icon } from "@landrik/ui";
```
## Usage

All Icon components can be meaningful or presentational icons.

<Canvas withSource="open">
  <Story name="base">
    <Icon name="watch" size={30} />
  </Story>
</Canvas>


<Canvas withSource="open">
  <Story name="iconWithLabel">
    <div style={{}}>
      <ul style={{
        display: 'flex',
        flexFlow: 'wrap',
        listStyle: 'none',
        margin: '0',
        padding: '0'
      }}>
        {Object.keys(icons).map((key) => (
          <li 
            key={key}
            style={{
              display:'inline-flex',
              minWidth: '120px',
              flexDirection: 'row',
              alignItems: 'center',
              flex: '0 1 16%',
              margin: '16px',
            }}
          >
            <span style={{
              width: '14px',
              height: '14px',
              marginRight: '16px',

            }}>
             <Icon name={key} aria-label={key} />
            </span>
            <p>{key}</p>
          </li>
        ))}
      </ul>
    </div>
  </Story>
</Canvas>
